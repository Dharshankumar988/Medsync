# Changelog

All notable changes to this project will be documented in this file.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]
- **Message Queues**: RabbitMQ/Celery integration for heavy AI processing.
- **Multi-Hospital Scaling**: Tenant architectures for distinct hospital networks.

## [1.0.0] - 2026-06-26
### Added
- **Monorepo Architecture**: Integrated Backend, Web, Mobile, and Blockchain.
- **Next.js Web App**: Full portal for Patients, Doctors, and Pharmacies.
- **Expo Mobile App**: Cross-platform native application with Secure Store.
- **DevOps**: Docker Compose, GitHub Actions CI/CD, Nginx reverse proxy.
- **AI Modules**: Patient RAG LLM Chat (Groq) and Doctor Computer Vision pipeline (YOLO/EfficientNet).
- **Blockchain**: Smart Contracts deployed on Polygon Amoy for immutable record metadata.

## [0.5.0] - 2026-05-15
### Added
- **FastAPI Backend Foundation**: JWT Auth, RBAC, SQLAlchemy models.
- **IPFS Integration**: Secure file hashing and storage protocols.

## [0.1.0] - 2026-04-01
### Added
- Initial Database schema and ER diagrams.
